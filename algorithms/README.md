# Reference MAR Algorithms

**The `reference_*.py` files in this directory use parallel-beam geometry
(v6.0.0 / ASTM WKXXXXX Rev 03)** and are **not compatible** with v7.0.0 fan-beam
sinograms (SID=570mm, SDD=1040mm, 720 angles).

Fan-beam (v7.0.0) reference algorithms live in the [`v7/`](v7/) subdirectory —
see the status table below.

## Usage

The parallel-beam `reference_*.py` algorithms process sinograms generated by
`legacy/generator_v6_0_0.py` (parallel-beam); do not use them with datasets from
`generator_v7_0_0.py`. For v7.0.0 fan-beam datasets, use the references in
[`v7/`](v7/) instead.

To use the parallel-beam references with v6.0.0 datasets:
```bash
python algorithms/reference_nmar.py --input-dir ./astm_mar_ils_v6_dataset
```

## Porting to Fan-Beam

Porting these algorithms to v7.0.0 fan-beam geometry requires replacing the
`skimage.transform.radon/iradon` calls with fan-beam forward projection and
FBP from `mar_ils_core` or `generator_v7_0_0.py`.
