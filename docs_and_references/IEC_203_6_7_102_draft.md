# IEC 60601-2-44 Ed. 4 — Proposed Amendment

## §203.6.7.102 — Quantitative MAR Performance Evaluation

**Status:** Draft proposal for IEC SC 62B WG 30 consideration
**Date:** 2026-04-05
**Author:** Christopher D. Cocchiaraley (Consumer Member, ASTM F04; IEC SC 62B WG 30)
**Depends on:** §203.6.7.101 (MAR metadata recording — FDIS stage)
**References:** ASTM WKXXXXX Rev 04

---

### Context

IEC 60601-2-44 Ed. 4 §203.6.7.101 mandates that manufacturers record MAR metadata
in DICOM when a MAR function is active. This proposed §203.6.7.102 extends that
requirement to include quantitative performance evaluation of the MAR function's
effect on diagnostic task performance.

The clause is intentionally permissive: it mandates measurement and disclosure, not
a specific performance threshold. Acceptance criteria are the responsibility of the
RESPONSIBLE ORGANIZATION (national regulatory body).

---

### Proposed Clause Text

**203.6.7.102 Quantitative evaluation of METAL ARTIFACT REDUCTION performance**

**203.6.7.102.1** If the MANUFACTURER declares a METAL ARTIFACT REDUCTION function
per 203.6.7.101, the MANUFACTURER shall evaluate the effect of the METAL ARTIFACT
REDUCTION function on diagnostic task performance using a standardized quantitative
test method.

**203.6.7.102.2** The evaluation shall measure the signed difference in observer
performance between the METAL ARTIFACT REDUCTION enabled condition and the METAL
ARTIFACT REDUCTION disabled condition, using a signal-known-exactly,
background-known-statistically (SKE/BKS) detection task and a model observer.

NOTE 1 — ASTM WKXXXXX provides a suitable test method. The measurand is ΔAUC
(difference in area under the ROC curve), computed using a channelized Hotelling
observer on a standardized digital dataset.

**203.6.7.102.3** The test result shall include:

- (a) the signed difference in observer performance (e.g., ΔAUC) with confidence
  interval;
- (b) identification of the test method used;
- (c) identification of the standardized dataset and its integrity verification
  result (e.g., SHA-256 checksum);
- (d) all imaging parameters applied during evaluation, with explicit confirmation
  that only the METAL ARTIFACT REDUCTION enable/disable state differed between
  conditions.

**203.6.7.102.4** The test result shall be included in the ACCOMPANYING DOCUMENTS.

NOTE 2 — This clause does not establish a minimum performance threshold. The purpose
is to ensure that the effect of METAL ARTIFACT REDUCTION on diagnostic task
performance is measured, documented, and available for review by the RESPONSIBLE
ORGANIZATION.

NOTE 3 — A negative test result (indicating that the METAL ARTIFACT REDUCTION
function degraded diagnostic task performance) is a valid and reportable outcome
and shall not be suppressed.

**203.6.7.102.5** The quantitative evaluation shall be repeated whenever the METAL
ARTIFACT REDUCTION algorithm is modified in a manner that could affect diagnostic
task performance.

NOTE 4 — The MANUFACTURER should consider evaluating multiple clinical configurations
(e.g., different implant materials, anatomical regions) to characterize the METAL
ARTIFACT REDUCTION function across its intended use conditions, even though this
clause requires only one standardized evaluation.

---

### Rationale

**Why quantitative evaluation is needed:**

§203.6.7.101 requires that MAR be recorded in DICOM metadata, establishing
traceability. However, recording alone does not inform the RESPONSIBLE ORGANIZATION
about whether the MAR function helps or harms diagnostic performance. A
manufacturer could record "MAR applied" while the MAR algorithm degrades lesion
detectability — §203.6.7.101 would be satisfied, but patient safety would not be
served.

§203.6.7.102 closes this gap by requiring a standardized, objective measurement of
MAR's effect on a clinically relevant task (lesion detection). The measurement is
independent of the manufacturer's claims and provides a common basis for regulatory
review.

**Why no threshold:**

Performance thresholds are a regulatory policy decision, not a measurement standard
decision. Different RESPONSIBLE ORGANIZATIONS may set different thresholds based on
their risk-benefit frameworks. IEC provides the measurement mandate; national
regulatory bodies set acceptance criteria through their own guidance mechanisms.

This mirrors established practice in IEC 60601 series, where particular standards
define measurement methods and test procedures, while national regulatory bodies
establish acceptance criteria (e.g., maximum permissible dose limits, minimum image
quality metrics).

**Relationship to ASTM WKXXXXX:**

ASTM WKXXXXX defines the test method (how to measure ΔAUC). This IEC clause
defines the regulatory requirement (must measure and report). The two documents are
complementary and independently ballotable. The "e.g., ASTM WKXXXXX" language in
NOTE 1 allows this clause to proceed through IEC ballot before the ASTM standard
is finalised, with the normative reference hardened in a subsequent amendment.
